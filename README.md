## вљ™пёЏ Installation

### Clone Repository

```bash
git clone https://github.com/ashwani1121/T-CAB-Cab-Booking-Application.git
```

### Go to Project

```bash
cd T-CAB-Cab-Booking-Application
```

### Install Client Dependencies

```bash
cd client
npm install
```

### Install Server Dependencies

```bash
cd ../server
npm install
```

### Start Backend

```bash
npm run dev
```

```bash

npm start
```
